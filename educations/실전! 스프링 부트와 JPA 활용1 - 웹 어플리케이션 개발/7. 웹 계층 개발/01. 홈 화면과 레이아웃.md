# 홈 화면과 레이아웃

## 정리

- layout
  - include-style
    - 무식한 방법
    - 계속 include를 해줘야 합니다.
    - 새로운 메뉴를 만들때 include를 해줘야 합니다.
  - Hierarchical-style
    - 중복 코드를 줄일 수 있습니다.
  - springboot-devtools
    - html 파일 수정시 recompile만 하면 됩니다.
- static(css, js) 파일 반영
  - 해당 폴더 reload
  - 해당 프로젝트 Build 진행

## TIP

- 단축키 recompile : command + shift + 9
- 단축키 build : command + F9
