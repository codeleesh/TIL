# HTTP 응답 데이터 - 단순 텍스트, HTML

## 정리

- 단순 텍스트 응답
- HTML 응답
- HTTP API - MessageBody JSON 응답

- HTTP 응답으로 HTML을 반환할 때는 content-type을 `text/html`로 지정해야 한다.
