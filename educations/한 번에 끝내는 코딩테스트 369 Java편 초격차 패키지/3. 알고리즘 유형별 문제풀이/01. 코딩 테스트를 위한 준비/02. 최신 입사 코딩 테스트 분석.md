# 최신 입사 코딩 테스트 분석

- 주관적인 난이도
  - 문법을 알고 있다.
  - 알고리즘과 자료구조를 이해하고 있다.
  - 적확한 선택을 통해 문제를 풀 수 있다.
  - 코딩 테스트에서 정답률 10%쯤 되는 문제
  - 코테 수준의 지식만 사용하지만 풀이를 떠올리기 쉽지 않고 구현 난이도도 상당한 문제

## 필수 테크닉

- 정렬
- 문자열 처리
- DP
- Dijkstra
- BFS & DFS
- 완전 탐색
- 이분 탐색

## 높은 난이도로 나오는 알고리즘

- 배열에서의 DP
- Tree 에서의 DP
- 투 포인터
- 위상 정렬

## 자료구조

- HashMap
- Deque(Queue)
- Stack
- Priority Queue
- Dynamic Array
