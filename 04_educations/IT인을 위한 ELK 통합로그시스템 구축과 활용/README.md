# IT인을 위한 ELK 통합로그시스템 구축과 활용

- 기간 : 2022.04.01 ~ 2022.04.31
- 강의 : [멀티캠퍼스](https://lc.multicampus.com/s-core)
- 수강 내용 :
  - ELK 설치는 실제 서버 설치 가이드라서 조금 아쉬웠음
    - 실제 설치는 docker-compose를 이용해서 진행하였음
  - Elasticsearch 강의 버전은 7.2이지만 최근에는  버전 8까지 나와서 8로 설치해서 진행하였음
    - API Deprecated 된 부분들이 있어서 docs를 찾아보면서 진행하였음
  - 실습 전반적인 부분은 차례차례 진행하여서 도움이 많이 되었고 특히 kibana 실습을 직접 해볼 수 있어서 도움이 많이 되었음
  - 데이터 에제를 임포트해서 진행하는 부분은 너무 좋았고 이렇게 데이터를 생성하고 테스트할 수 있다는 것이 좋았음
  - 그외 윈도우-ELK 연동 예제 등 개발자로 살면서 과연 하게 될까? 느꼈던 부분들은 과감히 스킵하였음
